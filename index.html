HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is often not show to the user.
  • HTML is the foundation for a website (adding sturcture to the content of a webpage)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

CSS

  • There is three types of styling a webpage with CSS: inline CSS, CSS style sheet, and external CSS style sheet.
  • The most common wayt to style a webpage is with external CSS style sheet.
  • A .card can be used to add CSS to specific parts of and HTML

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "commit message": Used to add a message to a commit
  • git pull origin main: syncs the local branch with the remote branch in GitHub
  • git push origin "branch name": Pushes changes from working branch to main branch in GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a group of code that can be called on and executed.
  • JavaScript is case sensitive.
  • Go back and find a definition for if and then statements
  • Go back and define a for statement
  • Boolean is used for true/false statements.
